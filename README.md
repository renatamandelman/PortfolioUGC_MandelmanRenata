# Renata Mandelman Portfolio

Portfolio profesional para Proyecto Senior - Diseño Multimedia.

## Estructura

```
PortfolioUGC/
├── index.html          # Página principal
├── css/
│   └── styles.css   # Todos los estilos
├── js/
│   └── main.js     # Interactividad
├── assets/
│   └── cv-renata-mandelman.pdf  # Tu CV (reemplazá este archivo)
└── README.md
```

## Cómo personalizar

### 1. Videos
Reemplazá los iframes en `index.html` líneas 120-180 con tus videos reales:
- **YouTube**: Usá el embed ID
- **Instagram Reels**: Usá el URL del reel
- **TikTok**: Embed code desde TikTok

### 2. Proyectos
Editá las tarjetas en la sección `#proyectos` (líneas 185-240):
- Cambiá `[Nombre Proyecto 1]` por el nombre real
- Editá la descripción de estrategia
- Agregá screenshots en `project-card__image`

### 3. CV
Reemplazá `assets/cv-renata-mandelman.pdf` con tu CV real.

### 4. Redes sociales
Actualizá los links en `#contacto` (líneas 260-290).

## Deployment en Vercel

### Opción 1: CLI (rápido)
```bash
npm i -g vercel
vercel
```

### Opción 2: GitHub + Vercel
1. Subí este repo a GitHub
2. Importá el repo en vercel.com
3. Listo!

### Opción 3: Drag & Drop
1. Vas a vercel.com/drop
2. Arrastrás la carpeta PortfolioUGC
3. Vercel deploya automático

## Tech Stack

- **HTML5** semántico
- **CSS3** con Custom Properties
- **Vanilla JS** (sin frameworks)
- **Google Fonts**: Syne + DM Sans
- **Deploy**: Vercel (gratis)
